# MonadRefGraph - Viral Distribution Dashboard

A developer dashboard for configuring multi-level referral systems on Monad blockchain.

**[Live Demo](https://machoman3000.github.io/viral-distribution-dashboard/)**

## Features

- 3-step registration flow (Register → Configure → Integrate)
- Interactive payout configuration with real-time preview
- Retro terminal aesthetic
- Built with Next.js 16, React 19, and Tailwind CSS 4

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Tech Stack

- **Framework:** Next.js 16.1.6
- **UI:** React 19 + Tailwind CSS 4
- **Deployment:** GitHub Pages (static export)
